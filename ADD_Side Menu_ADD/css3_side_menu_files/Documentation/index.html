Overview

This menu can be integrated into your HTML pages easily. Here are the basic steps :

  • Make a safe copy before beginning
  • Copy the content of the file sidemenu.css or the file itself and link to it into your HTML.
  • Add the images and icons (and update all paths in the CSS and/or HTML if necessary).
  • Now you can grab the HTML content of the menu and paste it into your pages, the menu starts with the following element :
<div id="sidemenu_wrapper">

HTML Structure

The whole menu is an unordered list that can hold as many items as you want (as long as they fit into the page height). All versions use exactly the same markup except a couple of classes that define the alignment and the color scheme.

In the example below, the menu will stick to the left edge of the browser and will use the light color scheme :

<div id="sidemenu_wrapper">

    <ul id="sidemenu" class="sidemenu_light sidemenu_left">


    </ul>

</div>

Each menu item come with a panel opened on mouse hover so from the previous code snippet, here's a first menu item with its panel :

<div id="sidemenu_wrapper">

    <ul id="sidemenu" class="sidemenu_light sidemenu_left">

        <li class="sidemenu_first"><span id="sidemenu_id"></span>
  
            <div class="sidemenu_container">
       

            </div>

        </li>

    </ul>

</div>

The above structure will create a menu item that requires an ID (sidemenu_id in this example, each one has a different icon styled in the CSS) with a container for the panel (sidemenu_container).

As a side note, the first menu item should have the class sidemenu_first and the last one the class sidemenu_last, it's simply used to fix borders.

Another option is to make the panel aligned with the top of the side menu, hover over the 3rd element in the default files to see that the contact form sticks to the top of the menu. For that, there's a single additional class to apply to the LI parent item, if we take the previous markup sample, we just add the top_panel class as following :

<div id="sidemenu_wrapper">

    <ul id="sidemenu" class="sidemenu_light sidemenu_left">

        <li class="top_panel"><span id="sidemenu_id"></span>
  
            <div class="sidemenu_container">
       

            </div>

        </li>

    </ul>

</div>

On the opposite, it is possible to align the panel with the bottom of the side menu just by using the class bottom_panel as following (and we apply it to the last element with the class sidemenu_last) :

<div id="sidemenu_wrapper">

    <ul id="sidemenu" class="sidemenu_light sidemenu_left">

        <li class="sidemenu_last bottom_panel"><span id="sidemenu_id"></span>
  
            <div class="sidemenu_container">
       

            </div>

        </li>

    </ul>

</div>

The last step is to give a dimension to the panel so you can add there your content. You'll need first a wrapper that will hold the columns. You have the choice between 3 sizes for the panels :

  • sidemenu_1col can hold 1 column and has a width of 155px
  • sidemenu_2col can hold 2 columns and has a width of 310px
  • sidemenu_3col can hold 3 columns and has a width of 465px

So let's make from the code above a 2-columns panel :

<div id="sidemenu_wrapper">

    <ul id="sidemenu" class="sidemenu_light sidemenu_left">

        <li><span id="sidemenu_id"></span>
  
            <div class="sidemenu_container">

                <div class="sidemenu_2col">
                
                2 COLUMNS CONTAINER

                </div>

            </div>

        </li>

    </ul>

</div>

Now we can add either a single column taking the whole container width or 2 small columns. Let's add 2 small columns :

<div id="sidemenu_wrapper">

    <ul id="sidemenu" class="sidemenu_light sidemenu_left">

        <li><span id="sidemenu_id"></span>
  
            <div class="sidemenu_container">

                <div class="sidemenu_2col">
                
                    <div class="col_1">

                    COLUMN 1

                    </div>

                    <div class="col_1">

                    COLUMN 2

                    </div>

                </div>

            </div>

        </li>

    </ul>

</div>

CSS Files and Structure

The file sidemenu.css contains all the necessary code to make the menu functionnal. You may want to remove the first part of the CSS that styles the page for demonstration purposes.

If you need to edit any part of the CSS, just scroll to the corresponding label. The file is divided into the following parts :

00. Page Styling (Removable)
01. Side Menu Bar
02. Side Menu Panels
03. Side Menu Columns
04. Side Menu Typography
05. Side Menu Gallery
06. Side Menu Form Elements
07. Light Color Scheme
08. Dark Color Scheme
