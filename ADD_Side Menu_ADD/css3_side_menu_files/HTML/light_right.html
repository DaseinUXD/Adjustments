CSS3 Side Menu

This brand new CSS3 Side Menu is an easy solution to add any kind of content always accessible from anywhere on your pages. It comes into 4 versions : left or right aligned, light or dark.

The basic setup to use it requires the sidemenu.css stylesheet, the images (icons) and all the HTML markup of the menu (all the content starting with the #sidemenu_wrapper element). Swithing from a version to another one is a simple matter of classes, just switch between them and you can change the alignment as well as the color scheme.

Integrating and customizing this menu requires at least a basic knowledge of HTML and CSS, this is not a plugin ready to be uploaded to your Wordpress or Joomla site. All modifications have to be done in the source code.

Features

  • Fancy side menu
  • Easy to integrate and customize
  • Can hold (almost) any kind of HTML element
  • 3 columns grid
  • Works only with CSS
  • Left or right aligned
  • Light and dark color schemes
  • Clean and attractive element
  • 100% valid HTML/CSS markup
  • No CSS hacks for IE
  • Uses some CSS3 effects
  • Cross-browser tested

It has been tested with all the major browsers and works from IE7+. All tests have been done with native applications running under a virtual machine to get accurate results. Please keep in mind that some CSS3 features don't work with old browsers like rounded corners. For more informations, please visit findmebyIP.
