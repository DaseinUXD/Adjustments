D) Some Editing in CSS - top

  • To change the color of menu, find this code background-image: color(whatever) in the "ul#menu" selector. You can use this link for color selection.
  • To change the font-size, font-family and line-height of the menu, find this code font: 14px/1.5 ColaborateRegular, calibri, tahoma, sans-serif; in the #menu li a selector.
